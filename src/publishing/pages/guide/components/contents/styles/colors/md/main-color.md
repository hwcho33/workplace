# &#x1F4CC; NS Main Colors

## &#x1F4C1; description

- 서비스의 브랜드 컬러를 대표하는 NSmall 서비스의 메인 컬러입니다.
- Main Color와 Semantic Color는 코어 컬러로 모든 NSmall 서비스 및 경험에서 기본 작업 색상으로 사용되는 대표 색상입니다.

## &#x1F4C1; variables

```scss
--red-900: #5c0101;
--red-800: #820101;
--red-700: #a00303;
--red-600: #c20404;
--red-500: #dc0a0a;
--red-400: #f20e0e;
--red-300: #f63846;
--red-200: #ff6873;
--red-100: #ffbbc0;
--red-50: #ffe5e7;
--ns-red: var(--red-500); // main red
--ns-coral-200: var(--red-300);
--ns-coral-100: #fc5460;
```
